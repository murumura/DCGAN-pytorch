# DCGAN-pytorch
Yet another pytorch implementation of Deep Convolutional Generative Adversarial Networks
# Dependencies
- Python 3.7.7
- PyTorch version: 1.6.0
- CUDA version: 10.1

# Usage
- Training using Mnist dataset
```sh
sh train_mnist.sh
```
